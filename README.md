# 🕐 WorldClock — 跨平台多时区桌面时钟

> 与全球客户沟通时，一眼看清对方的时间。约跨国会议时，一秒找到最佳时段。

## 🎯 为什么需要这个工具？

你是否经常遇到这些问题？

- 🤝 和纽约客户约会议，要算半天对方现在是几点
- 🌍 团队横跨 4 个时区，找不到所有人都合适的时间
- 📱 打开浏览器搜时区再换算，流程烦琐

**WorldClock** 是一个常驻桌面的小工具，**打开电脑就能看到**各时区时间，再也不用打开浏览器查。

---

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🕐 **6 时区同屏显示** | 本地时间 + 5 个自定义国际时区，一目了然 |
| 🖱 **自由拖拽** | 无边框窗口，可拖到桌面任意位置 |
| 📌 **窗口置顶** | 一键置顶，随时对照 |
| 🌓 **深色/浅色/自动主题** | 三档循环切换，自动跟随系统 |
| 📅 **会议助手** | 自动计算各时区工作重叠区间，推荐最佳会议时间 |
| ⚙ **自定义工作时间** | 为每个时区独立设置上班/下班时间 |
| 📏 **字体大小可调** | 小/中/大三档，适配不同屏幕 |
| 💾 **位置记忆** | 关闭后自动恢复上次窗口位置 |

## 🚀 快速开始

### 下载安装包

从 [Releases](https://github.com/yuhan0304/world-clock/releases) 页面下载最新版本：

| 平台 | 下载文件 |
|------|----------|
| Windows | `WorldClock-Setup-1.0.0.exe` |
| macOS | `WorldClock-1.0.0.dmg` |

### 或从源码运行

```bash
# 克隆仓库
git clone https://github.com/yuhan0304/world-clock.git
cd world-clock

# 安装依赖
npm install

# 启动
npm start
```

### 一键启动（Windows）

双击 `start.bat` 即可自动检查依赖并启动。

### 快捷操作

| 操作 | 效果 |
|------|------|
| 拖拽窗口任意位置 | 移动窗口 |
| 点击 🌙 / ☀️ | 主题切换（自动 → 深色 → 浅色） |
| 点击 ⭐ | 切换窗口置顶 |
| 点击 ✎（卡片右上角） | 替换该时区 |
| 点击 📅 | 展开/收起会议助手 |
| 点击 ⚙（会议面板） | 自定义各时区工作时间 |
| 点击 A A A | 切换字体大小 |

## 🛠 本地打包

```bash
# Windows 安装包
npm run build:win

# macOS 安装包（需在 macOS 上运行）
npm run build:mac
```

打包后的文件在 `dist/` 目录。

## 📂 项目结构

```
world-clock/
├── main.js              # Electron 主进程
├── preload.js           # 安全桥接层
├── package.json         # 项目配置
├── start.bat            # Windows 一键启动
├── build.bat            # 一键打包脚本
├── .github/workflows/   # GitHub Actions 自动打包
└── src/
    ├── index.html       # 主页面
    ├── css/style.css    # 蓝白商务极简主题
    └── js/
        ├── main.js      # 渲染逻辑
        └── timezone.js  # 时区工具库
```
